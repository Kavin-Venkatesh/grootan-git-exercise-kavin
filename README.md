
# Grootan Git Exercise

A simple project for learning Git workflows and version control practices.

## Overview

This project serves as a hands-on exercise for understanding Git fundamentals, including branching, commits, merging, and collaboration.

## Features

- Basic project structure
- Git workflow exercises
- Route demonstrations

## Routes

- **`/api/health`** - Server health check
- **`/api/users/user/:id`** - Fetch a user by ID
- **`/api/users/update-user/:id`** - Update user details by ID
- **`/api/users/delete-user/:id`** - Delete user by ID
- **`/api/users/all-users`** - Fetch all users
- **`/api/users/create-user/`** - Create a user with user details

## Getting Started

1. Clone the repository.
2. Navigate to the project directory.
3. Follow the exercises in order.

## Usage

```bash
npm install
npm run dev # For development
```
